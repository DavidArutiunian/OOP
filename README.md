### Installation

#### Getting all dependencies

```bash
mvn dependency:go-offline
```

#### Packaging JARs

```bash
mvn -Dmaven.test.skip=true package
```

### Running tests

```bash
mvn test
```

### Requirements

* Java 11
* Kotlin 1.3.11
* Maven 3.6.0
* Visual Studio 2019 (C++ 17)
